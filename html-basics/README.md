Welcome to your HTML learning journey! This folder contains a step-by-step guide to mastering the fundamentals of HTML (HyperText Markup Language).

## Table of Contents

1.  **[01_introduction.html](./01_introduction.html)**
    *   Basic Boilerplate (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
    *   Meta tags and `lang` attribute.
    *   HTML Comments.

2.  **[02_text_elements.html](./02_text_elements.html)**
    *   Headings (`h1` to `h6`).
    *   Paragraphs (`p`) and line breaks (`br`).
    *   Text Formatting (`strong`, `em`, `mark`, `small`, `del`, `sub`, `sup`).

3.  **[03_lists.html](./03_lists.html)**
    *   Unordered lists (`ul`).
    *   Ordered lists (`ol`) with different types.
    *   Description lists (`dl`, `dt`, `dd`).
    *   Nested lists.

4.  **[04_links_and_images.html](./04_links_and_images.html)**
    *   Anchor tags (`a`) for internal and external links.
    *   `target="_blank"` for opening in new tabs.
    *   Images (`img`) with `src` and `alt` attributes.
    *   Using images as links.

5.  **[05_tables.html](./05_tables.html)**
    *   Table structure (`table`, `tr`, `th`, `td`).
    *   Semantic table sections (`thead`, `tbody`, `tfoot`).
    *   Cell spanning (`colspan`, `rowspan`).

6.  **[06_forms.html](./06_forms.html)**
    *   Form container (`form`) and submission (`action`, `method`).
    *   Input types: `text`, `password`, `email`, `number`, `date`, `radio`, `checkbox`, `file`.
    *   Labels and accessibility.
    *   Dropdowns (`select`, `option`) and multi-line text (`textarea`).

7.  **[07_semantic_html.html](./07_semantic_html.html)**
    *   Importance of semantic tags.
    *   Layout tags: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`.
    *   Non-semantic tags: `div` and `span`.

8.  **[08_attributes_and_meta.html](./08_attributes_and_meta.html)**
    *   Global attributes: `id`, `class`, `title`, `style`, `hidden`, `data-*`.
    *   SEO Meta tags.
    *   Character Entities (e.g., `&lt;`, `&copy;`).

## How to use these files?
Simply open any of the `.html` files in your web browser (Right-click -> Open with -> Chrome/Edge/Firefox). 

Each file is extensively commented with explanations to help you understand "what" each tag does and "why" it is used.
